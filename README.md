# Ready-To-Roomba

Are you ready to Roomba? Are you ready to change your life? Your life will be dust-free and crust-free. Pick up a Roomba today!

## Get Started

Download into your local drive:

- `npm install`
- `createdb grace-shopper`
- `npm run seed`
- `npm run start:dev`
